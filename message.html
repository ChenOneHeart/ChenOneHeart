Message 消息提示 ​

常规的消息提示弹框，从屏幕上方向下弹出消息提示。可以设置弹框类型、持续时间、关闭按钮、弹框间距。

基础用法 ​

<script setup>
import { h } from "vue";
import { createMessage } from "@/components/Message/methods.ts";
import Button from "@/components/Button/Button.vue";
const open = () => {
  createMessage({ message: "hello world", duration: 0, type: "success" });
};

const openVn = () => {
  createMessage({
    message: h("p", "使用VNode方式生成message")
  });
};
</script>
<template>
  <Button @click="open">message 基础用法</Button>
  <Button @click="openVn">message VNode语法</Button>
</template>

Type弹窗类型 ​

有成功（success）、警告（warning）、消息（info）、错误（error）四种弹框类型，默认值为info。

<script setup>
import { createMessage } from "@/components/Message/methods.ts";
import Button from "@/components/Button/Button.vue";
const openInfo = () => {
  createMessage({ message: "Info 类型的弹框", duration: 0, type: "info" });
};
const openSuccess = () => {
  createMessage({ message: "success 类型的弹框", duration: 0, type: "success" });
};
const openWarning = () => {
  createMessage({ message: "warning 类型的弹框", duration: 0, type: "warning" });
};
const openDanger = () => {
  createMessage({ message: "Danger 类型的弹框", duration: 0, type: "danger" });
};
</script>
<template>
  <Button @click="openInfo">info 类型</Button>
  <Button @click="openSuccess">success 类型</Button>
  <Button @click="openWarning">warning 类型</Button>
  <Button @click="openDanger">danger 类型</Button>
</template>

ShowClose关闭按钮 ​

使用showClose属性设置是否添加关闭按钮。

<script setup>
import { h } from "vue";
import { createMessage } from "@/components/Message/methods.ts";
import Button from "@/components/Button/Button.vue";
const open = () => {
  createMessage({ message: "默认无关闭按钮", duration: 0, type: "info" });
};
const openHideClose = () => {
  createMessage({ message: "隐藏关闭按钮", duration: 0, showClose: false, type: "info" });
};
const openShowClose = () => {
  createMessage({ message: "开启关闭按钮", duration: 0, showClose: true, type: "info" });
};
</script>
<template>
  <Button @click="open">默认状态</Button>
  <Button @click="openHideClose">隐藏关闭按钮</Button>
  <Button @click="openShowClose">开启关闭按钮</Button>
</template>

Duration持续时间 ​

<script setup>
import { createMessage } from "@/components/Message/methods.ts";
import Button from "@/components/Button/Button.vue";
const open = () => {
  createMessage({ message: "默认持续3000ms", type: "info" });
};
const open0 = () => {
  createMessage({ message: "弹框一直保留", duration: 0, type: "success", showClose: true });
};
const open1000 = () => {
  createMessage({ message: "持续 1000ms", duration: 1000, type: "warning" });
};
const open2000 = () => {
  createMessage({ message: "持续 2000ms", duration: 2000, type: "danger" });
};
</script>
<template>
  <Button @click="open1000">持续 1000ms</Button>
  <Button @click="open2000">持续 2000ms</Button>
  <Button @click="open">默认状态</Button>
  <Button @click="open0">不关闭按钮</Button>
</template>

Offset弹框间距 ​

<script setup>
import { h } from "vue";
import { createMessage } from "@/components/Message/methods.ts";
import Button from "@/components/Button/Button.vue";
const open = () => {
  createMessage({ message: "hello world", duration: 0, type: "success" });
};
const open50 = () => {
  createMessage({ message: "hello world", duration: 0, type: "success", offset: 50 });
};
const open100 = () => {
  createMessage({ message: "hello world", duration: 0, type: "success", offset: 100 });
};
</script>
<template>
  <Button @click="open">默认间距</Button>
  <Button @click="open50">间距 50px</Button>
  <Button @click="open100">间距 100px</Button>
</template>
