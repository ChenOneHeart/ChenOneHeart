Icon 图标 ​

常规的图标，使用fontawesome进行二次封装的图标组件，通过icon属性可以传入图标参数，支持fontawesome所有参数。 （fontawesome的详细用法参阅https://docs.fontawesome.com/web/use-with/vue）。

基础用法 ​

图标的基本使用方法。

<script setup>
import Icon from "@/components/Icon/Icon.vue";
</script>
<template>
  <Icon icon="arrow-up" color="#0e7a0d" />
  <Icon icon="arrow-down" color="#0e7a0d" />
  <Icon icon="arrow-left" color="#0e7a0d" />
  <Icon icon="arrow-right" color="#0e7a0d" />
</template>
<style scoped>
.vc-icon {
  margin-right: 10px;
}
</style>

Type类型 ​

图标的基本使用方法。

<script setup>
import Icon from "@/components/Icon/Icon.vue";
</script>
<template>
  <Icon icon="fa-coffee" type="success" />
  <Icon icon="house" type="info" />
  <Icon icon="cloud" type="warning" />
  <Icon icon="comment" type="danger" />
</template>
<style scoped>
.vc-icon {
  margin-right: 10px;
}
</style>
